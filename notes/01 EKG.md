# 01 EKG

- use tick marks to see if regular/irregular
- pattern beating: dropped/extra beats
- Sinus: regular, P before QRS, QRS after P
- Wenckeback: P gets longer between first and last, but can be same between each beat

# Hypertrophy

- LVH: AS, AR
- LVH+LAE: MR
- LAE: MS
